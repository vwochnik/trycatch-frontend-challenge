About the Application

This application has been developed for the TryCatch front-end coding challenge to show my software development competences, particularely in the area of front-end development.

  • The look and feel of the user interface has been kept as simple as possible on purpose to keep complexity at bay and to avoid using heavy UI frameworks such as Bootstrap or Foundation for this simple application.

  • Many best practices are employed such as the DRY and single responsibility principles by using serivces and directives provided by the AngularJS framework.

  • The application has been developed using a simplified version of Scrum where a branch including an issue has been created for every user story.
